# 🎯Part·1 实战演练

## 1. 题目内容

> <font color = red>**小红书**</font>是一个生活方式分享社区，用户可以在上面发现和分享美妆、时尚、旅行等领域的笔记和视频。
> 通过[小红书](https://www.xiaohongshu.com/explore)App你可以分享你的生活，同时也可以在这里认识到更多朋友，了解各种奇闻趣事。

本次现场编程，需要各团队在Android手机通过[Autox.js](http://doc.autoxjs.com/)来使用小红书，尽可能实现更多的交互功能。 也可以使用其他类似工具进行实现。
>[Autox.js下载](http://doc.autoxjs.com/)

题目分为四个部分：

- **基础题目**
  1. 打开小红书APP，进入某个直播间。
  2. 编辑文本，进行自动发帖。
- **中等题目**
	3. 小红书自动滑屏切换直播间，快捷点赞、关注。
	4. 小红书直播间自动点赞功能，进入任意一个直播间进行若干点赞。
- **进阶题目**
	5. 小红书直播间自动发评论功能，进入任意一个直播间发布若干评论。

## 2. 得分规则

- 根据提交的项目材料，评测组将按照以下评分规则进行评测。

- 本部分（即**现场提交**部分）满分**100分**，本部分得分**乘以80%后**计入得分。

- **基础题目（30分）**
  
  1. 自动打开小红书APP，选择一个自己感兴趣且符合社会主义核心价值观的直播间进入。**（13分）**
  2. 编辑文本，进行自动发帖。**（17分）**
  
- **中等题目（45分）**
	3. 小红书自动滑屏切换直播间，快捷点赞、关注。（本题将按实现程度、方式等进行评分）
	>3.1 自动滑屏切换直播间。**（12分）**
	>3.2 快捷点赞与关注。**（13分）**  
  4. 自由设计小红书直播间自动点赞功能，进入任意一个直播间进行若干点赞。
	>4.1 点赞数量无上限，越多越好。（注意点赞频率，防止死机）**（10分）**
	>4.2 点赞速度能够实时控制，按实现程度、实现方式、控制界面外观等评判得分 **（10分）**
- **进阶题目（共25分）**
	5. 小红书直播间自动发评论，进入任意一个直播间发布若干评论。
   >5.1 实现静态评论：发布的评论为不可修改，每条评论**各不相同**，数量越多越好，正能量向。**（15分）**
	>5.2 实现动态评论：发布的评论可以实时输入进行修改。**（10分）**

 
**以上题目最终解释权归任课教师所有。**
